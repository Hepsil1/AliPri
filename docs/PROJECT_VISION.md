# 🏅 СчётДетей — Видение проекта

## Миссия
Избавить руководителя спортивного клуба от ручного учёта посещений и абонементов.
Заменить бумажный журнал со «звёздочками» на удобное веб-приложение.

## Кто пользователь?
- **Руководитель клуба** — основной и единственный пользователь
- Ведёт 4 группы, ~45 спортсменов
- Работает с телефона на тренировке и с компьютера дома

## Ключевые боли (что решаем)

| # | Боль | Решение |
|---|------|---------|
| 1 | Ручной подсчёт звёздочек (занятий) | Автоматический счётчик с визуальным прогрессом |
| 2 | Забыл кому отправить напоминание | Автоматические уведомления при истечении абонемента |
| 3 | Путаница с разовыми и абонементными | Раздельный учёт с чётким статусом каждого |
| 4 | Заморозка при болезни — сложно отследить | Кнопка «Заморозить» с историей |
| 5 | Списание без присутствия | Отдельный статус «Не был, но списано» |

## Принципы дизайна
- **Mobile-first** — основное использование на телефоне во время тренировки
- **Один клик** — отметка посещения за минимум действий
- **Наглядность** — цветовая индикация остатка абонемента (зелёный → жёлтый → красный)
- **Оффлайн-готовность** — работа на локальном сервере без интернета

## Метрики успеха
- Время на отметку группы: **< 30 секунд** (vs 5+ минут вручную)
- Ноль пропущенных уведомлений по абонементам
- Полная история посещений каждого спортсмена
